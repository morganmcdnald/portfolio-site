A Selection of my Recent Works



SmolScreen Web App (Complete)

Description:

This is a Full-Stack app I created that acts as a sort of clone of the 'Letterboxd' website and app. Letterboxd is a sort of social media that is used by film-fans to keep track of movies they have seen, keep a wishlist and write reviews.

I started off creating the UI designs in Figma before moving to development. These designs can be found on my Behance using the link below.

For development, I created the web app using the Python framework Django. I also used Bootstrap for the development of the front-end interface, which I customised. For the database of the site I used PostgreSQL. For the movie data, I made use of the Movie Database API.


Technologies Used:

Django Python HTML5 CSS3 JavaScript Bootstrap


Note: Since the site is hosted on Heroku's free tier, load times will likely be longer than usual.
